JavaScript + WebGL Chip-8 Emulator

@author Ayden Diel

Controls

Pong

  • 1: left up
  • q: left down
  • 4: right up
  • r: right down

Snake

  • w: up
  • a: left
  • s: down
  • d: right

Space Invaders

  • q: left
  • w: shoot
  • e: right

About this Project

This is an emulator for the Chip-8 virtual machine that I wrote for fun. It's written in pure JavaScript and WebGL.

All of the source code is available here!
